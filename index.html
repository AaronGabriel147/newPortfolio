The Stoics

The Stoics is a fullstack web app that I built from scratch. This is an ancient philosophy app which allows users to create, read, update, and delete (CRUD) quotes that exist on a fully functional Node Express PostgreSQL database .

www.thestoics.com

Water My Plants

Water My Plants is a fully responsive web app which allows users to securely log in with a hashed password and a JSON Web Token. Users are then able to access various React routes which allows them to catalog their plants and watering instructions.

www.watermyplants.com
